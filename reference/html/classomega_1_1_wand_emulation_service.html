Detailed Description

Emulates a VR system wand using mouse and keyboard.

The mouse is used to generate a ray into the scene. The keyboard wasd keys are mapped to (Left, Right, Up, Down) buttons. NOTE: This service depends on events generated by MouseService and KeyboardService. MouseService and KeyboardService need to be instantiated for this service to work correctly. This service consumes all mouse and keyboard events, so they won't be available to the application.
